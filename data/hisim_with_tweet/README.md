# 简要说明

- event_macro_e1/2.pkl 为用于宏观评测的对应event的1000个用户的真实推文数据，key为用户名，每个用户的推文按照时间分为了14个阶段, 例如roe_macro_e2\[user1\]\[0\]即为roe的event2中user1在第0个时间步发布的推文, 每个时间步的总体立场标注结果放在了该时间步对应的第一条数据里；
- event_micro.pkl 为采样出来用于微观（个体）评测的用户真实推文数据，不区分时间段，单条推文就为一个独立样本。由于真实推文的content/stance/behavior的标注放到和模拟推文（的标注）一起在后面评测的地方做了，所以这里没有这些标签，可以根据文章附录B.3给的prompt让GPT标注一下。

注：请勿将此数据分发到任何地方
